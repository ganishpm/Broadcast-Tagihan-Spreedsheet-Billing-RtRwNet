# Broadcast-Tagihan-Spreedsheet-Billing-RtRwNet
Mengelola tagihan  pelanggan menggunakan spreedsheet. Saya gunakan untuk tagihan internet rt rw net billing

## Kegunaan
1. Broadcast pesan  tagihan ke pelanggan secara otomatis melalui whatsapp
2. Broadcast pesan informasi
3. Broadcast pesan terima pembyaran secara otomatis
4. Menggunakan 3 jenis provider API Whatasapp ( Wablas, Kirimi.id, Weagate)
5. Menghitung sisa tagihan, jika pelanggan tidak membayar full
6. Menghitung tagihan yang tertunggak sebelumnya

## Masih Proses perkembangan
1. Akan ada monitoring PPPOE ( monitoring yg lain akan menyusul jika ini berhasil )
2. Melakukan disable otomatis jika melewati tanggal tagihan
3. Dll ( semoga bisa mengimbangi billing diluar sana )

## Alasan ini dibuat
- Disarankan penggunaan jika customer belum 100 keatas karena akan repot mengurus banyak data
- saya saat ini mengurus data sekitar 50an, dari masing-masing server
- Alasan tidak menggunakan STB dan billing yg gratis. Karena saya mengurus beberapa server dan perlu biaya lagi untuk pengeluaran stb tersebut hehe
